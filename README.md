# Local Demo

## Requirements

You need to install:
* NodeJs: [Node.js](https://nodejs.org/)
* NPM: [NPM](https://npmjs.org/)

## Clone the repo and install the packages

    $ git clone https://github.com/OthLah001/latelier-tenis-backend.git
    $ cd latelier-tenis-backend
    $ npm i

## Run the server

    $ npm start
