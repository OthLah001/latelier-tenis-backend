# Local Demo

## Requirements

You need to install:
* NodeJs: [Node.js](https://nodejs.org/)
* NPM: [NPM](https://npmjs.org/)

## Clone the repo and install the packages

$ git clone https://github.com/OthLah001/latelier-tenis-frontend.git
$ cd latelier-tenis-frontend.git
$ npm i

## Run the server

$ npm start
